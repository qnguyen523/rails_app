# README

# Tech Stack
* **Site:** Ruby on Rails, HAML
* **UI:** jQuery, BootStrap, CSS.
* **Database:** PostgreSQL
* **Unit Tests:** rspec

# System dependencies
* Ruby version: 2.7.8p225
* Rails version: 7.0.8.4
* PostgreSQL version: 14.12

# How to run the application
* bundle install
* rails s

# How to run the test suite
bundle exec rspec
